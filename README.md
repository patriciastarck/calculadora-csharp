# Calculadora em C#

Este é um projeto para praticar lógica de programação e condicionais em C#.

## Funcionalidades

- Recebe um número inteiro do usuário do usuário.
- Permite escolher entre 4 operações básicas:
  - Soma (+)
  - Subtração (-)
  - Multiplicação (*)
  - Divisão (/)
  - Recebe o segundo número inteiro do usuário.
- Exibe o resultado no console.
